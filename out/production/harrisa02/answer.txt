The modulo operator (%) returns the remainder produced by the division of two numbers.
For example, if 5 / 2 is 2r1, then 5 % 2 = 1.

2.
What is the result of 8 % 3? If 8 / 3 = 2r2, then 8 % 3 = 2.

3.
The set of possible results of n % 5, where n is an integer, is 0 through 4.

4.
The set of possible results of n % m, where n and m are integers, is 0 through (m - 1).
